개인정보처리방침

확장 프로그램 이름: Side Panel Clock

개발자: Swyoon Labs

최종 수정일: 2026년 7월 21일

1. 개요

Side Panel Clock은 Chrome 사이드 패널에 시계, 세계 시계, 타이머, 뽀모도로 타이머를 표시하는 Chrome 확장 프로그램입니다.

본 확장 프로그램은 사용자의 개인정보를 존중합니다. 설정과 상태는 사용자의 브라우저에 로컬로 저장되며, 개인정보를 수집·전송·판매·공유하지 않습니다.

2. 수집하는 정보

Side Panel Clock은 사용자로부터 개인정보를 수집하지 않습니다.

다음 데이터가 사용자의 브라우저에 로컬로 저장될 수 있습니다.

  • 표시 환경설정 (테마 모드, 강조 색상, 시계 페이스, 12/24시간 형식, 초 표시, 아날로그 시계 켜기/끄기)
  • 사용자가 추가한 세계 시계 도시 목록
  • 타이머 및 뽀모도로 상태와 설정 (지속 시간, 세션 횟수)

본 확장 프로그램은 다음 정보를 수집하지 않습니다.

  • 이름
  • 이메일 주소
  • 비밀번호
  • 결제 정보
  • 위치 정보
  • 브라우징 기록
  • 웹사이트 콘텐츠
  • 분석(애널리틱스) 데이터
  • 추적 식별자

3. 사용자 데이터의 이용 방식

로컬에 저장된 환경설정과 상태는 시계, 세계 시계, 타이머, 뽀모도로 기능을 제공하기 위한 목적으로만 사용됩니다. 해당 데이터는 광고, 분석, 프로파일링 또는 기타 관련 없는 목적으로 사용되지 않습니다.

4. 데이터 저장

환경설정과 상태는 Chrome Storage API를 사용하여 저장됩니다. 이 데이터는 사용자의 브라우저 환경에 남아 있으며, Swyoon Labs 또는 제3자가 운영하는 외부 서버로 전송되지 않습니다. Side Panel Clock은 완전히 오프라인으로 동작합니다.

5. 데이터 공유

Side Panel Clock은 사용자 데이터를 제3자에게 판매, 대여, 이전, 공개 또는 공유하지 않습니다.

6. 원격 코드 및 외부 서비스

Side Panel Clock은 원격 코드 실행을 사용하지 않으며, 어떠한 데이터도 외부 서버나 제3자 서비스로 전송하지 않습니다.

7. Chrome 권한

  • sidePanel — Chrome 사이드 패널에 시계 인터페이스를 표시합니다.
  • storage — 환경설정, 세계 시계 도시, 타이머/뽀모도로 상태를 Chrome에 로컬로 저장합니다.
  • alarms — 패널이 닫혀 있어도 타이머와 뽀모도로 완료가 정확하게 울리도록 예약합니다.
  • notifications — 타이머가 종료되거나 뽀모도로 단계가 바뀔 때 데스크톱 알림을 표시합니다.

8. 사용자 제어

사용자는 확장 프로그램 내에서 설정과 도시를 변경하거나 삭제할 수 있습니다. 또한 확장 프로그램을 삭제하거나 Chrome에서 확장 프로그램의 저장 데이터를 지우면 로컬에 저장된 모든 데이터를 제거할 수 있습니다.

9. 아동 개인정보 보호

Side Panel Clock은 만 13세 미만 아동을 대상으로 하지 않습니다. 본 확장 프로그램은 아동으로부터 개인정보를 고의로 수집하지 않습니다.

10. 본 개인정보처리방침의 변경

본 개인정보처리방침은 확장 프로그램의 기능이나 데이터 처리 방식이 변경될 때 수정될 수 있습니다. 변경 시 수정된 "최종 수정일"과 함께 게시됩니다.
