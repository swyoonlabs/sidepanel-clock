# Side Panel Clock — Privacy Policy

**Extension Name:** Side Panel Clock
**Developer:** Swyoon Labs
_Last updated: 2026-07-21_

## 1. Overview

Side Panel Clock is a Chrome extension that shows a clock, world clock, timer, and Pomodoro
timer in the Chrome Side Panel.

We respect user privacy. This extension is designed to store its settings and state locally
and does **not** collect, transmit, sell, or share personal information.

## 2. Information We Collect

Side Panel Clock does not collect personal information from users.

The extension may store the following data **locally in the user's browser**:

- Display preferences (theme mode, accent color, clock face, 12/24-hour, seconds, analog on/off)
- The list of world-clock cities you add
- Timer and Pomodoro state and settings (durations, session count)

The extension does **not** collect:

- Names
- Email addresses
- Passwords
- Payment information
- Location information
- Browsing history
- Website content
- Analytics data
- Tracking identifiers

## 3. How User Data Is Used

Locally stored preferences and state are used only to provide the extension's clock, world
clock, timer, and Pomodoro functionality. The data is not used for advertising, analytics,
profiling, or any unrelated purpose.

## 4. Data Storage

Preferences and state are stored using Chrome's Storage API. This data remains in the user's
browser environment and is not sent to any external server operated by Swyoon Labs or any
third party. Side Panel Clock works fully offline.

## 5. Data Sharing

Side Panel Clock does not sell, rent, transfer, disclose, or share user data with third parties.

## 6. Remote Code and External Services

Side Panel Clock does not use remote code execution and does not send any data to external
servers or third-party services.

## 7. Chrome Permissions

- **sidePanel** — display the clock interface in Chrome's Side Panel.
- **storage** — save preferences, world-clock cities, and timer/Pomodoro state locally in Chrome.
- **alarms** — schedule timer and Pomodoro completions so they fire accurately even when the panel is closed.
- **notifications** — show a desktop notification when a timer ends or a Pomodoro phase changes.

## 8. User Control

Users can change or remove their settings and cities within the extension. Users may also
remove all locally stored extension data by uninstalling the extension or clearing the
extension's storage data in Chrome.

## 9. Children's Privacy

Side Panel Clock is not directed to children under the age of 13. The extension does not
knowingly collect personal information from children.

## 10. Changes to This Privacy Policy

This Privacy Policy may be updated when the extension's features or data handling practices
change. Any updates will be posted with a revised "Last Updated" date.
