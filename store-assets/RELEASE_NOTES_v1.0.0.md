# Side Panel Clock v1.0.0 — 시간을 늘 곁에 두는 사이드바 시계

> 브라우저 사이드 패널에 딱 붙어 있는 시계, **Side Panel Clock**의 첫 출시(1.0.0)를 소개합니다.
> 이번 버전의 핵심은 한 문장으로 **"곁눈질 한 번으로 시간·세계시각·집중을 모두"** 입니다.

---

## 🕐 한눈에 보기

| | 내용 |
|---|---|
| 🕐 **항상 보이는 시계** | 큰 디지털 + 아날로그 시계가 패널 상단에 늘 고정 |
| 🌍 **세계시계** | 주요 도시 20곳의 현지 시각·날짜·시차를 한눈에 |
| ⏲️ **타이머** | 원형 진행 링 + 프리셋(1·3·5·10·25분), 종료 시 알림·소리 |
| 🍅 **뽀모도로** | 집중 ↔ 휴식 자동 순환, 오늘 세션 카운트, 시간 커스터마이즈 |
| 🌐 **8개 언어** | 브라우저 언어에 맞춰 자동 현지화 |

---

## 🕐 1. 항상 보이는 시계 — 이 앱의 중심

사이드 패널의 강점은 **"다른 일을 하면서 계속 곁눈질로 볼 수 있다"** 는 점입니다.
그래서 큰 시계를 패널 **상단에 항상 고정**해 두었습니다. 탭을 옮기거나 새 창을 열 필요 없이,
브라우징하는 내내 시간이 한 글자만큼 떨어진 거리에 있습니다.

- **디지털 + 아날로그** — 취향에 맞게 아날로그 시계를 켜고 끌 수 있습니다.
- **12/24시간 · 초 표시** — 버튼 한 번으로 전환.
- 날짜·요일·오전/오후는 **브라우저 언어에 맞춰 자동 포맷**됩니다.

---

## 🌍 2. 세계시계 — 시차 계산은 이제 그만

원격 협업, 해외 가족·친구, 다른 시간대와의 약속. 매번 시차를 계산하지 않아도 됩니다.

- 주요 도시 20곳 중 원하는 곳을 추가하면 **현지 시각·날짜·현지 대비 시차**가 함께 표시됩니다.
- 필요 없으면 언제든 삭제. 도시명은 **현재 언어로 자동 표기**됩니다(Seoul / 서울 / ソウル / 首尔 …).

---

## ⏲️ 3. 타이머 — 남은 시간이 눈에 보이는 원형 링

- **원형 진행 링**이 카운트다운에 맞춰 줄어들어, 남은 시간이 직관적으로 보입니다.
- **프리셋**(1 · 3 · 5 · 10 · 25분) 또는 시·분·초를 **직접 입력**.
- 끝나면 **소리 + 데스크톱 알림** — 패널을 닫아 두어도 정확히 알려줍니다.

---

## 🍅 4. 뽀모도로 — 집중과 휴식을 자동으로

25분 집중, 5분 휴식… 검증된 시간 관리 기법을 그대로 담았습니다.

- 집중 → 짧은 휴식 → … → 4번째마다 긴 휴식으로 **자동 순환**.
- **오늘 완료한 집중 세션 수**를 카운트(🍅), 자정에 초기화.
- 집중·휴식·긴 휴식 길이를 **원하는 대로** 설정.
- 패널이 닫혀 있어도 백그라운드에서 계속 순환하고 단계 전환을 알립니다.

---

## 🌐 5. 8개 언어 — 어디서든 내 언어로

영어(기본)·한국어·일본어·중국어(간체)·스페인어·프랑스어·독일어·포르투갈어(BR).
UI 문구는 물론 **날짜·시각·도시명까지** 브라우저 언어에 맞춰 자동으로 바뀝니다.
