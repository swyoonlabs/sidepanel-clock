# Screenshots

블로그·문서·스토어에서 참조하는 스크린샷을 보관하는 폴더입니다.
블로그 포스트(`store-assets/blog-post-*.html`)가 이 폴더의 이미지를 GitHub raw URL로 불러옵니다.

| 파일 | 내용 |
|------|------|
| `spc-1-hero.png` | 히어로 — 상단 시계 + 세계시계 (라이트) |
| `spc-2-world.png` | 세계시계 (다크 테마) |
| `spc-3-timer.png` | 타이머 — 원형 진행 링 (라이트) |
| `spc-4-pomodoro.png` | 뽀모도로 — 진행 링 + 세션 카운트 (다크) |

> 스토어 업로드용으로 1280×800 크기에 맞춘 버전은 [`../store-assets/`](../store-assets/)에
> `store-1-*.png` 형식으로 따로 둡니다.
>
> 두 폴더의 이미지는 모두 [`../tools/make_store_assets.py`](../tools/make_store_assets.py)로 생성합니다.
